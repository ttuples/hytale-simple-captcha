# SimpleCaptcha for Hytale

SimpleCaptcha was originally a [Minecraft](https://modrinth.com/mod/simple-captcha) mod I made for fun, and now I've recreated it in Hytale for practice.

![](https://i.imgur.com/0TjKV2i.png)

## Disclaimer

This mod is intended for **joke purposes only** and should not be used as a security measure. It is designed to add a fun challenge for your friends, not to protect servers from unauthorized access.

## Commands

- `/captcha`: Triggers a captcha challenge for the player executing the command. (alias of /simplecaptcha)

- `/captcha --player=Player Name`: Triggers a captcha challenge to the specified player.

## Installation

To install SimpleCaptcha, follow these simple steps:

1. Download the SimpleCaptcha mod file from the releases section.
2. Place the downloaded `.jar` file into your servers `mods` folder.
